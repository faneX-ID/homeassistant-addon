# Changelog

All notable changes to this add-on will be documented in this file.

## [0.1.0] - 2025-12-13

### Added
- Initial release of faneX-ID Home Assistant Add-on
- Support for both SQLite and PostgreSQL databases
- Ingress support for seamless Home Assistant integration
- Developer mode to use latest main branch
- Database reset functionality with warning translations
- Microsoft Entra ID (SSO) support
- Automatic release version detection (including pre-releases)
- Renovate compatibility for automatic version updates
- Multi-language support (English and German)
- Configurable logging levels
- Auto-generated secret key support
- Demo mode option
- **HACS (Home Assistant Community Store) support**
- GitHub Actions CI/CD pipeline
- Automated multi-architecture Docker builds

### Features
- Downloads and builds faneX-ID from GitHub releases
- Builds frontend and backend in a single container
- Persistent data storage
- Health check endpoint
- Comprehensive configuration options
